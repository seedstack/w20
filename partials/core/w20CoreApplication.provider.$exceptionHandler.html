$exceptionHandler
service in module w20CoreApplication

Description

Replaces the default exception handler of AngularJS. It collapses similar errors (same type, same message, same stack) and emits an event (w20.core.application.error-occurred). This event can be used to display an error report.

Events

  • w20\.core\.application\.error-occurred

    This event is emitted on the root scope after an error has occurred. It can be used to display an error report.

    Type:

    emitted

    Target:

    root scope
    Parameters
    ParamTypeDetails
    allErrorsError[]

    The list of all errors that occurred since the last event.
