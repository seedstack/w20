w20CoreEnv

Description

This module provides W20 base features and abstraction from its running environment. Development environment is automatically set when global debug mode is active and no environment is specified.

Configuration

 "env" : {
     // Explicitly set the environment type (dev, test, preprod and prod are recognized special values).
     "type" : "the environment name"
 }
