SecurityExpressionService
service in module w20CoreSecurity

Description

The SecurityExpressionService provides an API to evaluate a security expression.

Methods

  • evaluate(expression, locals)

    Evaluates the specified security expression to a boolean.

    Parameters
    ParamTypeDetails
    expressionString

    The security expression.

    localsObject

    The local variables accessible in the expression.

    Returns
    Boolean

    True if the specified expression evaluates to true, false otherwise.
