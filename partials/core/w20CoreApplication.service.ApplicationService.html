ApplicationService
service in module w20CoreApplication

Description

The ApplicationService provides access to global application properties.

Methods

  • normalizeRouteName(path, separator)

    This method takes a route path and normalize it into a unique name. When multilevel paths (i.e. paths containing slashes) are normalized the separator parameter is used to build the name (default separator is the dot '.').

    Parameters
    ParamTypeDetails
    pathString

    Route path to normalize.

    separatorString

    The separator used to build the normalized name.

    Returns
    String

    The normalized route name.

Properties

  • applicationId

    This id can be used to disambiguate between multiple W20 applications when necessary.

    Returns
    String

    the unique id of the application.

  • homePath

    Returns
    String

    the path of the home route.
