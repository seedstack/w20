NavigationService
service in module w20CoreUI

Description

This service provides an api for dealing with routes hierarchy in the application. It is used in themes to build the tree menu for instance.

Methods

  • buildLink(path)

    This method take a route path and build a suitable link depending on pretty URLs mode. Path is prefixed by the hashbang if application is in legacy mode. Path is stripped of starting slash if application is in HTML5 mode.

    Parameters
    ParamTypeDetails
    pathString

    Route path to build link from.

  • computeSubTree(parentMenuTree)

    Given a parent tree this function will compute the underlying subtree. This function is used to determine the deeper level of themes sidebar tree in a recursive way.

    Parameters
    ParamTypeDetails
    parentMenuTreeObject

    The parent tree

    Returns
    Object

    The subtree of a parent tree

  • expandedRouteCategories()

    Return the list of the opened categories in a tree menu

    Returns
    Array

    A list of the expanded route categories i.e the categories opened in a route menu of themes

  • refreshNavigation()

    Refresh the routes hierarchy. This function is used on security event such as login/logout to refresh the available routes for instance.

  • routeTree()

    Returns the route hierarchy (i.e organized by categories).

    Returns
    Object

    A map of routes hierarchically organized

  • routesFromCategory(category)

    Returns the list of all visible routes from a given category name.

    Parameters
    ParamTypeDetails
    categoryString

    The category from which to obtain all the available routes

    Returns
    Array

    A list of the visible routes of the supplied category

  • routesFromCategory(route)

    Routes can be hidden (see security in w20CoreApplication). This function return the visibility state of a given route.

    Parameters
    ParamTypeDetails
    routeObject

    The route to check visibility of

    Returns
    Boolean

    The visibility status of the route

  • topLevelRouteCategories()

    The top level categories are the one that appear first in a menu before any tree are opened. This function return these top level categories.

    Returns
    Array

    A list of the top level categories

  • topLevelRoutes()

    Routes that are not part of a category are top level routes. Returns a list of these routes.

    Returns
    Array

    A list of the top level routes
